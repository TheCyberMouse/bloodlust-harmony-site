# Home page slideshow

Drop 1920x1080 screenshots here (.jpg, .png, or .webp). They show in the
slideshow below the hero on the home page automatically, sorted by filename.

Naming: prefix with a number to control order, then a short description.
The description becomes the image's alt text.

    01-massed-assault-on-the-castle.jpg
    02-undead-vs-paladin-front-line.jpg
    03-night-siege.jpg

After adding or removing files, run publish-site.cmd in the repo root
(or commit + push) to deploy.
