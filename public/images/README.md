# General image library

Anything that is not a slideshow screenshot or a brand asset: concept art,
GIFs, images for lore/guide/devlog posts, press kit extras.

Reference from prose pages or site code as /images/<filename>, e.g.
an image named orc-concept.jpg is served at:

    https://bloodlust-harmony-site.vercel.app/images/orc-concept.jpg

Keep filenames lowercase-with-dashes. After adding files, run
publish-site.cmd in the repo root to deploy.
